# Images Folder

Add your images here for the landing page.

## Recommended Images:

1. **hero-background.jpg** (or .png)
   - The blurred stage/audience background image
   - Recommended size: 1920x1080 or larger
   - This will be used as the background behind the speaker

2. **speaker-image.jpg** (or .png)
   - The main image of Brian Moses on stage
   - Recommended size: 800x1200 or similar portrait orientation
   - This is the prominent image on the right side of the hero section

## How to Use:

1. Add your image files to this folder (`public/images/`)
2. The code will automatically reference them as `/images/your-filename.jpg`
3. Supported formats: .jpg, .jpeg, .png, .webp

## Example:
- `public/images/hero-background.jpg` → Use as `/images/hero-background.jpg`
- `public/images/speaker-image.jpg` → Use as `/images/speaker-image.jpg`
